#### Project setup

##### If you want to run it yourself you should use those comands:

```
npm install
```

##### DevServer can be used to quickly develop an application

```
npm run start
```

##### Compiles and hot-reloads for development

```
npm run dev
```

##### Compiles and minifies for production

```
npm run prod
```
